How to Play

Objective: Move the red block all the way to the exit at the right of the board.

You need to clear the path and move blocks out of the way.

Vertical blocks can only move up and down. Horizontal blocks can only move left and right.

Each board has an Optimal # of Moves and a Par to represent difficulty

Try to unblock the red block in as few moves as possible!
